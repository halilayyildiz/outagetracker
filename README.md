Outage Tracker
=============

Mobile iOS appllication to track electricity outages.

Features
-------------------
- Basic outage listing.
- Calendar view for outages.
- Push notificitons for actual outages.
- Reminder for planned outages.

 - iOS mobile application (ver 6+)
